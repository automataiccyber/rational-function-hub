What are Rational Functions?

A rational function is defined by a rational fraction, which is an expression of the form f(x) = P(x) / Q(x) where both P(x) and Q(x) are polynomials and Q(x) ≠ 0.

Domain and Range

The domain includes all real numbers except where Q(x) is zero, because division by zero is undefined. The range is influenced by the asymptotes and the behavior of the function as x approaches positive or negative infinity.

Asymptotes

Asymptotes are lines that the graph approaches but never touches. There are vertical asymptotes, occurring where Q(x) = 0, and horizontal asymptotes, indicating the function's end behavior as x approaches infinity or negative infinity.

Zeros and Holes

Zeros occur where the numerator P(x) = 0. Holes occur where both the numerator and denominator are zero, indicating a point of discontinuity that is not an asymptote.

Example of a Rational Function

Example: f(x) = (x2 - 1) / (x2 - 4)

Domain: All real numbers except where x2 - 4 = 0, which gives x = ±2. Therefore, the domain is all real numbers except x = 2 and x = -2.

Range: Determined by the values that f(x) can take.

Asymptotes: Vertical asymptotes at x = 2 and x = -2. No horizontal asymptote because the degree of the numerator and denominator are the same.

Zeros: The zeros of the function are where x2 - 1 = 0, giving x = ±1.

Holes: There are no holes because the numerator and denominator do not share any common factors that cancel each other out.
